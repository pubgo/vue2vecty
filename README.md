# vue2vecty
translate vue to vecty

- 一个把vue转换为vecty的工具
- A tool to convert vue to vecty

## help
### build

```bash
git clone https://github.com/pubgo/vue2vecty.git
go build main.go

# build vue template
./main b 

# build vue template and monitor
./main b -m 
```

### create component
```bash
./main component button/button-group 
./main page v1/home 
```
